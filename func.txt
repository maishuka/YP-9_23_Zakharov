 метод это набор операторов которые выполняются вместе
для создания метода необходимо задать его имя, определить список параметров и тело метода
для вызова метода другого класса необходимо, что бы он был объявлен как паблик
вызов осуществляется по выводу классов и методов
если метод не void, то необходимо вернуть значение соотвествуещего типа, оператор return останавливает выполнение метода и передает управление вызвавшему данный метод оператору
каждый метод имеет набор своих локальный переменных они видны только в нем и при завершении работы метода уничтожаются для того что бы переменные были видны в нескольких методов класса
необходимо объявить их полями вне метода но внутри класса, для не void методов небходимо возваращать значение каждой "путь выполнения" метода должен заканчиватся оператором return, для воид методов оператор return не обязателен
использование параметров, параметры позваляют передавать информацию из одного метода в другой при объявлении метода можно задать список его параметров, если список пустой, то это означает метод не имеет параметров
c# существует 3 варианта предеи параметров:по значению, по ссылке 
выходные параметры где по значению (изменения значения параметра в методе не влияет на значение в вызвавшим в методе)
по ссылке (необходимо указывать ключевое слово ref)
в c# позваляет использовать мехонизм передачи списка параметра 
модификаторы доступа это ключевые слова которые задают объявленный уровень доступности члена или типа 
семь уровней специальных вызможностей модификаторов доступа:
public доступ не ограничен 
protected доступ ограничен содержащим классом 
internal доступ ограничен текущей сборкой 
protected internal 
privated доступ ограничен содержащим типом 
private protected 
file объявленный тип отображается только в текущем исходном файле
